komono
======

小物（こもの）って言葉、可愛くありませんか？

### include
お得なC言語用ライブラリ詰め合わせパック	
`make`すると共有ライブラリができます
```shellsession
% cd include
% make
```

### bsodnt4
WindowsNT Workstation 4.0のOS Loader直後からBSODが出るまでを再現したやつ	
ncursesが必要です
```shellsession
% gcc bsodnt4.c -lncurses -o bsodnt4
```
